---------------------------------------nginx简单逻辑----------------------------------------
nginx的配置中不支持if条件的逻辑与&& 逻辑或|| 运算 ，而且不支持if的嵌套语法，否则会报下面的错误：nginx: [emerg] invalid condition。
可以用变量的方式来间接实现。
ep：
set $flag 0;
if ($uri ~ ^/thumb/[0-9]+_160.jpg$){
 set $flag "${flag}1";
}
if ($arg_tagId = 1){
 set $flag "${flag}1";
}
if ($flag = "011"){
 echo "www.test.com";
}
